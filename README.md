# Rpifo-cli

## English
BASH script for real-time raspberryPi monitoring
### How to use : 
```bash
# Quick install
curl -L -o rpifo.sh https://raw.githubusercontent.com/debmus/rpifo-cli/master/en/rpifo.sh
chmod +x rpifo.sh
bash rpifo.sh
```
